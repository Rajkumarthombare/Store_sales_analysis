# Store Sales Analysis Dashboard
Project Objective
The goal of this project was to analyze retail store data to identify sales trends, customer preferences, and payment behaviors to help drive data-informed business decisions.

Dataset Overview
Total Sales: ₹1211.16K

Total Orders: 5,000

Unique Customers: 5,000

Average Rating: 3.78/5

Key Insights & Findings
Category Dominance: Electronics accounts for the majority of sales, followed by Footwear and Sports.

Age Group Distribution: The "31-50" age bracket is the primary revenue driver (₹506K), followed closely by the "50+" group (₹470K).

Gender Analysis: Male customers contribute significantly more to the total sales volume (81%) compared to Female customers (19%).

Seasonal Trends: Sales peak during the Spring season, while Autumn shows the lowest activity, suggesting a need for targeted marketing during off-peak months.

Operational Efficiency: 39% of customers are repeat buyers, indicating a stable customer retention rate.

Technical Skills Applied
Data Cleaning: Processed raw data using Excel to ensure accuracy.

Pivot Tables: Summarized large datasets for quick analysis.

Dashboard Design: Created an interactive UI using Slicers for Age, Gender, Season, and Payment Method.

Data Visualization: Used Donut charts, Bar charts, and Trend lines to communicate insights effectively.
